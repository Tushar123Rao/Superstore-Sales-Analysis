# Project Overview
This project performs a complete **Exploratory Data Analysis (EDA)** on the 
Superstore Sales Dataset to uncover key business insights about sales, 
profit, discounts and customer behaviour.

# Dataset
- **Source:** Kaggle - Superstore Dataset
- **Rows:** 9,994
- **Columns:** 21
- **Features: Sales, Profit, Discount, Region, Category, Sub-Category, 
Order Date, Ship Date

# Key Business Insights

1. Technology is the highest selling Category
2. West Region generates the highest Profit
3. Sales have grown consistently every year
4. November & December have highest monthly sales (Holiday Season!)
5. Higher Discounts are directly hurting Profits
    - Negative Correlation between Discount & Profit
6. Top selling products are mostly Technology items
